# MINUS LAG

**Minus Lag** is a high-performance network optimization tool designed to reduce ping, jitter, and packet loss for competitive gaming.

## Features

- **Direct Route Optimization**: Optimizes network paths to game servers.
- **Live Server Detection**: Automatically detects the active game server IP usage for real-time ping monitoring.
- **UDP Multipath**: Sends duplicate UDP packets to mitigate packet loss (configurable).
- **Game Launcher**: Integrated game launcher with auto-launch capabilities for supported titles (Valorant, etc.).
- **Process Priority Management**: Automatically adjusts game process priority for better performance.

## Installation

Download the latest installer from the [Releases](../../releases) page.

1.  Download `MINUS LAG Setup.exe` (NSIS) or `MINUS LAG.msi`.
2.  Run the installer.
3.  Launch the application as Administrator (required for network driver access).

## Usage

1.  Select your game from the dashboard.
2.  Choose your region.
3.  Toggle "Auto-Launch" if desired (🚀 icon).
4.  Click **Start Optimization**.
